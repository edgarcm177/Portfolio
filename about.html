HTML

HTML is an easy markup language to pick up that is used to make webpages online. To begin a HTML document all you need to do is start of with a doctype tag that tells a computer what type of document it is. Then you make the head tag followed by a closing head tag and then you make a body tag. In this body tag is where most of the content of the webpage you are making will go. Most body tags start with headers which desginate what the webpage is about. Then this is usually followed by text, buttons, input fields, or graphs depnding on the purpose of the webpage. Then after your done making content you simply close your body and then html tags. All of this serves as a base which then CSS builds upon.


CSS

CSS is what makes a website appealing and it can be done in various ways. For instance there is inline styling which is when you apply a style to a tag such as the color or font size by typing out the style attribute on the tag. Then theres internal where you make a style tag usually in the head and apply a style to the entire document. Then the most common form of CSS is external styling where you link a seprate document with all your CSS to your HTML document. This is to help seperate CSS from HTML and to have CSS apply to multiple webpages. With CSS there are so many things you can do like the formatting, styling, and even animations. Together with HTML they make up most of what we see on webpages.


JavaScript

JavaScript is very interesting because when you get past the basics you get to actually see what your code is doing. For instance when I made a number guessing game. To achieve this I first made a button in my HTML document to call up a function I would make in JS and an input field to guess. Then in my JS file I made a function that used an already built in JS function to generate a number. Then with if statements I checked if the number in the input field was correct and if it was incorrect I checked if it was higher or lower. Depending on what statements it satisfied it would output text below the button that said either your correct, guess lower, or guess higher. Being able to see what my code did in real time was satisfying in the end because it gave me a sense of accomplishment after completing this difficult task.
